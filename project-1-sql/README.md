# 📊 Análisis Financiero de Mercados Internacionales (SQL)

## 📌 Contexto
El objetivo del proyecto es analizar el desempeño financiero de diferentes mercados internacionales para identificar oportunidades de mejora en la rentabilidad.

## 🔍 Análisis
Se utilizaron consultas SQL para:
- Unir múltiples tablas (JOINs)
- Calcular métricas financieras
- Filtrar y agrupar datos

## 📈 Resultados
- Identificación de mercados con mayor rentabilidad
- Comparación de ingresos y costos
- Evaluación de margen y ROI

## 🛠️ Herramientas
- SQL
